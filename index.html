Finance Dashboard Backend

Zorvyn API landing page

This repository contains the finance data processing and access control backend for the assignment. It provides role-based user access, financial record CRUD, dashboard summaries, validation, and a lightweight mock auth model for local and hosted demos.

Framework Node.js with Express and SQLite.
Auth Mock header-based auth via x-user-id.
Roles viewer, analyst, and admin permissions enforced in middleware.
Docs OpenAPI and Swagger UI are included in the backend.
